# University of Washington Concrete Canoe Structural Analysis
Written by Samarjit Singh, Heerod Sahraei

## Explanation of Modules

Run BuoyantLoad.m to generate Paddler Loads, Buoyancy, Self-Weight
Run MomentShear.m to generate Moment and Shear Diagrams for loadcase.
Run RaceStands.m to generate Moment and Shear Diagrams for a race stand loadcase.

Incomplete: Run CableDesign.m to find optimal post-tensioned cable force for given eccentricity. 

## Goals
- Incorporation of Post-tensioning design into code
- Live diagrams and streamlined UI
- Organized codebase
