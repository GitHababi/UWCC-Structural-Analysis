# University of Washington Concrete Canoe Structural Analysis
Written by Samarjit Singh, Heerod Sahraei

## Explanation of Modules

Open the live script Main.m, input all relevant information, then hit Run!
Each Section of the code is as follows

Structural Analysis:
1. Physical Constants and Material Properties
2. Paddler Properties and Race Stand Positions
3. Force Analysis
4. Buoyancy Solver
5. Shear, Moment, and Internal Stresses
6. Punching Shear

Reinforcement Design:

7. Cable Design
8. DCR and Final Values

Sandbox.m and Plot.m contain additional helpers for other information 
and graphing.
